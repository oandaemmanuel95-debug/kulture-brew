publish (quick) Option A — GitHub Pages (free) 1. Create a GitHub repository named `kuture-roast`. 2. Add the files (index.html, about.html, products.html, contact.html) and a folder `assets` with styles.css and any images (hero-coffee.jpg). 3. In repo Settings → Pages, choose branch `main` and folder `/ (root)` and save. 4. Your site will be published at: `https://.github.io/kuture-roast/` (use HTTPS). Example: `https://manu.github.io/kuture-roast/`. Option B — Netlify (drag & drop) 1. Zip the site files (root with index.html) and drag the zip to Netlify Drop (https://app.netlify.com/drop). 2. Netlify will publish to a random subdomain like `https://sleepy-kangaroo-12345.netlify.app/` — you can set a custom domain. Option C — Vercel 1. Create a Vercel account and import the repo. 2. Deploy and receive a `https://.vercel.app` URL. Notes: - Replace the WhatsApp placeholder number `254700000000` with your phone number. - To use `http://` specifically (non-SSL) is discouraged; hosts above provide `https://` by default. If you must use `http://` with a custom domain, configure DNS and hosting provider accordingly. I recommend HTTPS. ---- END OF DOCUMENT ----
